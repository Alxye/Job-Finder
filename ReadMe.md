# Job-Finder
thinkphp5框架搭载的带后台管理功能的招聘类网站
- 部分素材来源于互联网，侵删，联系邮箱：944652593@qq.com
- 说明：nav 和 footer作用于前台界面所有模块，因此把他们放在Common模块下对应输入关键字搜索功能在nav.html中实现展示，在各个控制器index方法中实现搜索。
